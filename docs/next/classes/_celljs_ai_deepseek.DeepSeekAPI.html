  • embed<T>(embeddingRequest): Promise<ResponseEntity<EmbeddingResponse>>

  • Creates an embedding vector representing the input text or token array.

    Type Parameters

    • T

      Type of the entity in the data list. Can be a string or list of tokens (e.g., integers). For embedding multiple inputs in a single request, you can pass a list of strings or a list of lists of tokens.

    Parameters

    • embeddingRequest: EmbeddingRequest<T>

    Returns Promise<ResponseEntity<EmbeddingResponse>>

    A promise that resolves to an embedding response containing a list of embeddings.

    Example

    embed(List.of(List.of(1, 2, 3), List.of(3, 4, 5)));
